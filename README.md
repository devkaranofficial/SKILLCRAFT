**Problem Statement**

Predict house prices using:

* Square Footage
* Number of Bedrooms
* Number of Bathrooms

**Dataset**

* Ames Housing Dataset

**Features Used**

* GrLivArea
* BedroomAbvGr
* FullBath

* SalePrice

**Workflow**

* Dataset Loading
* Feature Selection

* Linear Regression

* Model Evaluation
