# House Price Prediction using Linear Regression

## SkillCraft Technology - Machine Learning Internship

### Task 01

This project implements a Linear Regression model to predict house prices based on housing characteristics from the Ames Housing Dataset.

## Problem Statement

Build a machine learning model that predicts house prices using:

* Square Footage
* Number of Bedrooms
* Number of Bathrooms

## Dataset

Ames Housing Dataset

Files Used:

* train.csv
* test.csv

## Features

* GrLivArea (Above Ground Living Area)
* BedroomAbvGr (Number of Bedrooms)
* FullBath (Number of Bathrooms)

## Target Variable

* SalePrice

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-Learn

## Workflow

1. Load Dataset
2. Select Features
3. Split Data into Training and Testing Sets
4. Train Linear Regression Model
5. Generate Predictions
6. Compare Actual vs Predicted Values
7. Evaluate Performance

## Evaluation Metrics

| Metric   | Value            |
| -------- | ---------------- |
| MAE      | 35,788.06        |
| MSE      | 2,806,426,667.25 |
| R² Score | 0.6341           |

## Results

The Linear Regression model achieved an R² Score of 0.6341, indicating that the selected features explain a significant portion of the variation in house prices.
